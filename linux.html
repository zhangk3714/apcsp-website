Name Description Example
echo Echo the STRING(s) to standard output echo hello world would print hello world
date Display the current time in the given FORMAT, or set the system date date would print Tue 03 Dec 2019 12:28:20 PM PST
hostname Hostname is used to display the system's DNS name, and to display or set its hostname or NIS domain name hostname would print rpi18
arch Print machine architecture arch would print armv71
uname Print certain system information uname would print Linux
who Print information about users who are currently logged in who would print sp1a pts/0 2019-12-03 12:24 (172.19.32.55)
id Print user and group information for the specified USER, or (when USER omitted) for the current user id would print uid=1001(sp1a) gid=1001(sp1a) groups=1001(sp1a),997(gpio)
man man is the system's manual pager. Each page argument given to man is normally the name of a program, utility or function man man would display the manual page for the command man
yes Repeatedly output a line with all specified STRING(s), or 'y' yes would display y y y ...
sleep Pause for NUMBER seconds sleep 3 would make the program stop for 3 seconds
